How to play:
1) Install XNA 4.0 redist (xnafx40_redist.msi)
2) Start Game
3) Enter username (first time only or change it manually in settings.ini)
4) Enjoy


movement:	WASD
Shoot:		LMB + RMB (special attack on cooldown)
boost:		ctrl

All abilities except movement costs energy, energy is recharged while playing.
Survive against the mobs in the arena and get the highest score!
